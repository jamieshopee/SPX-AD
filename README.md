# SPX AD 電子版位管理器

## 如何啟動

- 控制台：雙擊 `launch/啟動 AD 管理器（Chrome）.command`
- 編輯器：雙擊 `launch/啟動編輯器_*.command`

## 主要入口

- `index.html`：控制台主入口
- `canvas.html`：Banner Render Engine
